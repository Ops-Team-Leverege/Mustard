Log Level Usage Recommendations
Now that you have log levels, here's how to use them in events.ts:

debug - Verbose details for development:

logger.debug('Meeting resolution attempt', { hasMeetingRef, regexResult, llmResult });
logger.debug('Thread context lookup', { threadTs, foundContext: !!priorInteraction });
info - Key milestones (keep current usage):

logger.info('Slack event received', { text: text.substring(0, 100), isReply });
logger.info('Control Plane completed', { intent, contract });
logger.info('Request completed successfully', { totalTimeMs });
warn - Recoverable issues:


logger.warn('Thread context lookup failed', { threadTs, error: err.message });
logger.warn('Progress message already sent', { totalTimeMs });
error - Actual failures (keep current usage):

logger.error('Pipeline error', err, { errorType, errorCode });
